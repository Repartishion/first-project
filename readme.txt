Привет! Я только начал изучание репозиториев. Вот что я знаю:

cd - перейти в директорию
- ~ домашняя директория
- .. прошлая директория
- / корневая директория

ls - просмотреть директорию
cat - прочитать txt файл
clip - скопировать

mkdir - создать папку
touch - создать файл

rm - удалить файл
rmdir - удалить папку
rm -a - удалить папку и файлы в ней

git status - просмотреть информацию о последнихъ обновлениях файла
git add - подготовить файл(ы) к коммиту
git commit - сделать коммит
git push - отправить изменения на github
